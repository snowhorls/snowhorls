setup-email-and-password[gitutorial]

git config --global user.email "email" setup email
git config --global user.name "username" setup username
git config --global core.editor vim setup default editor

remove the --global flag to set it locally instead

remote-repos2, 3

git remote -v lists the remote servers you have configured
git remote add <shortname> <url> adds a remote repo like github
git remote add origin https://github.com/username/repo

using-a-credential-helper4

this tells git to store your credentials unencrypted on the disk so you dont have to log in everytime you push

git config --global credential.helper store   

see git-credential-store and git-credentials

using-git

when you want to add, commit, and push changes

git commit -a -m "commit-message" && git push <remote> <branch>

where remote is a remote repository and branch is your branch to push
